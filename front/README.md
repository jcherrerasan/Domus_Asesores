# Front

Frontend estático y minimalista para el CRM.

## Uso rapido

1. Arranca el backend en `http://localhost:8080` o `http://localhost:8082`.
2. Sirve esta carpeta con un servidor estatico.

```powershell
cd E:\domus_asesores\front
python -m http.server 5500
```

3. Abre `http://localhost:5500`.
4. Si hace falta, cambia la URL del backend en la parte superior.

## Incluye

- Alta y listado de usuarios
- Alta y listado de leads
- Alta y listado de clientes
- Alta y listado de propiedades
- Alta y listado de valoraciones
